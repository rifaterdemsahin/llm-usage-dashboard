The one thing that changes the whole comparison: Open Interpreter calls models through API keys, not chat subscriptions. So Claude Pro and Gemini Advanced don't power Open Interpreter on their own — you'd need separate API billing for each. GLM Coding Pro and DeepSeek are the natural fits because both expose Anthropic-format API endpoints OI can point at directly: GLM as a flat monthly rate, DeepSeek as near-free pay-per-token.

💸 Cheapest at volume

DeepSeek V4 Flash. Once a Playwright/Selenium script works, you don't need frontier reasoning to re-run it, and the per-token cost is negligible.

📊 Most predictable budget

GLM Coding Pro. Flat rate means a runaway agentic loop won't surprise you with a bill.

🛠️ Most reliable at writing the automation

Claude via API (Opus 4.8), since it still has the edge on long-running agent loops with tool use. Point OI at the Anthropic API, not your $20 Pro plan.

Caution: LinkedIn's User Agreement prohibits automated scraping, bots, and automated connection/messaging activity. Running automation against it risks account restriction or a permanent ban, and LinkedIn has pursued scrapers legally. The compliant route is LinkedIn's official Marketing/Talent Developer APIs rather than browser automation — model choice only matters once you've sorted out which path you're on.
