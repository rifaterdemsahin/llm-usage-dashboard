How to collect your usage numbers

The dashboard shows cost, requests, and tokens per provider. Here's where to get each one. Enter them under ⚙ Settings — values are saved to MongoDB Atlas and cached so the same query isn't sent repeatedly.

OpenRouter live (automatic)
1. Go to openrouter.ai → Settings → Keys and create an API key (sk-or-...).
2. Paste it into Settings → OpenRouter. The backend calls GET /api/v1/key to pull your credits used automatically.
3. For requests/tokens, see openrouter.ai/activity and enter them as the manual fallback.
Claude (Anthropic) manual
1. Open the Anthropic Console → Usage / Cost (or Billing).
2. Read your cost ($), requests, and input+output tokens for the period.
3. Enter them in Settings → Claude. (Anthropic's usage API needs an Admin key and can't be called from the browser, so these are entered manually.)
Gemini (Google) manual
1. In Google AI Studio or Google Cloud Console → Billing / APIs & Services, open the Generative Language API metrics.
2. Read your cost ($), request count, and token count.
3. Enter them in Settings → Gemini.

Settings

Enter API keys and/or usage numbers per provider. OpenRouter can be fetched live from your browser; Claude & Gemini block direct browser calls (CORS), so enter their numbers manually (or wire a backend proxy later).
